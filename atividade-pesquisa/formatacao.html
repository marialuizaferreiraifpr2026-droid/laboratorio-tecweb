Elementos HTML <small> HTML

O elemento HTML <small> define um texto menor. É usada para representar textos de "letras miúdas", comentários laterais, direitos autorais ou avisos legais, reduzindo o tamanho da fonte para torná-la menor que o texto ao redor. Embora reduza o tamanho visualmente, seu principal objetivo é semântico, indicando comentários secundários.
